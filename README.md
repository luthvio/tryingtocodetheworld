# tryingtocodetheworld
Just a beginner with big curiosity

This was my first coding project, and when I deployed it to GitHub Pages, I ran into an issue where around 80% of the CSS wasn't being applied and the JavaScript didn't work at all.

The first thing I did was check my project in VS Code to see if any files or code had been deleted. Everything looked fine. Then I opened my GitHub repository and checked the CSS and JavaScript files directly.

That's when I noticed something strange: the CSS file only contained about 10 lines of code, and the JavaScript file was completely empty. I checked the project folder on my computer to make sure, and surprisingly, the files there had the same issue.

After going back to VS Code, I realized that my code hadn't been fully saved. Some of the latest changes were still unsaved. I saved all the files again, pushed the project to GitHub once more, and finally everything worked as expected. The website looked exactly the same as the preview in VS Code.

That was my first troubleshooting experience, and thankfully the solution was pretty simple: always make sure your files are fully saved before pushing them to GitHub.
